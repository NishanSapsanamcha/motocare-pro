# MotoCare Pro

A full-stack web application built with the **PERN stack**:
- **PostgreSQL** (database)
- **Express.js** (backend framework)
- **React** (frontend)
- **Node.js** (runtime)

## Tech Stack

**Frontend**
- React

**Backend**
- Node.js + Express.js
- PostgreSQL
- JWT/Auth 
- Multer/Cloudinary

## Prerequisites

Make sure you have these installed:

- **Node.js** (LTS recommended)
- **npm** (comes with Node)
- **PostgreSQL** (running locally or via a hosted provider like Neon/Supabase/Railway)
